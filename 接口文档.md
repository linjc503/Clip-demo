# 接口文档

## 基础信息

- 基础 URL：`http://localhost:8000`
- API 文档：`http://localhost:8000/docs`

---

## 1. 添加文本

**接口**：`POST /add_text`

**请求体**：
```json
{
  "text": "商品名称"
}
```

**响应**：
```json
{
  "product_id": "a1b2c3d4",
  "text": "商品名称"
}
```

---

## 2. 添加图片

**接口**：`POST /add_image`

**表单数据**：
- `image`：图片文件（必填）
- `description`：商品描述（可选）

**响应**：
```json
{
  "product_id": "e5f6g7h8",
  "description": "商品描述"
}
```

---

## 3. 文本搜索

**接口**：`POST /search_text`

**请求体**：
```json
{
  "text": "搜索关键词"
}
```

**响应**：
```json
{
  "query": "搜索关键词",
  "results": [
    {
      "id": "xxx_image",
      "similarity": 0.95,
      "metadata": {
        "type": "image",
        "product_id": "xxx"
      }
    }
  ]
}
```

---

## 4. 图片搜索

**接口**：`POST /search_image`

**表单数据**：
- `image`：图片文件

**响应**：同文本搜索

---

## 5. 零样本分类

**接口**：`POST /classify_image`

**表单数据**：
- `image`：图片文件

**响应**：
```json
{
  "category": "Topwear",
  "confidence": 0.92,
  "all_categories": [
    {"category": "Topwear", "confidence": 0.92},
    {"category": "Bottomwear", "confidence": 0.05}
  ]
}
```

**支持类别**：Topwear、Bottomwear、Accessories、Footwear、PersonalCare

---

## 6. 智能客服

**接口**：`POST /chat`

**请求体**：
```json
{
  "message": "有什么好看的衬衫推荐吗？"
}
```

**响应**：
```json
{
  "reply": "根据您的需求，我为您找到以下衬衫商品..."
}
```

---

## 前端页面

访问 `http://localhost:8000` 可使用可视化界面，包含：
- 文本录入区域
- 图片录入区域
- 零样本分类区域
- 智能客服聊天区域
